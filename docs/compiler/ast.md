# Abstract Syntax Tree (AST)

The AST is the initial representation of the parsed Lala source code.

## Invariants
1. **Immutable**: Once generated by the parser, the AST must never be modified by downstream stages.
2. **Pure Syntax**: The AST only knows what the programmer wrote. It does not resolve types, variables, or intrinsics.
3. **Location Aware**: Every AST node must carry a `Span` representing its source location for diagnostics.

## Tree Structure
The AST is a deeply nested tree mirroring the grammar.
* `Program`
  * `FunctionDecl`
    * `Block`
      * `IfStatement`
        * `BinaryExpression`
          * `IdentifierExpression`
          * `NumberLiteral`
